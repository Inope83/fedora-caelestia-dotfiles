# 🌌 Caelestia Fedora Dotfiles

> Complete configuration for Caelestia on Fedora Linux

![Screenshot](screenshots/desktop.png)

## ✨ Features

| Category | Features |
|----------|----------|
| 🎨 **Dynamic Colors** | Material 3 theme follows wallpaper automatically |
| 🖼️ **Wallpaper** | Waypaper with swww animations (grow, wave, fade) |
| 🖥️ **Terminal** | Foot with auto-update colors from wallpaper |
| 🐟 **Shell** | Fish with starship prompt |
| 🎭 **Launcher** | Rofi/Wofi application launcher |
| ⚡ **System Info** | Fastfetch with custom Saturn logo |
| 📊 **Monitor** | Btop with Caelestia theme |
| 🔧 **Scripts** | update-borders, update-terminal, check-theme |

## 📋 Prerequisites

```bash
sudo dnf install hyprland foot fish fastfetch waypaper btop rofi-wayland
